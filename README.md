Ejercicio seminario angular

Pregunta de reflexión: Si el backend no utilizara .populate(), la organización solo devolvería los IDs de los usuarios asociados. Para mostrar el nombre de cada usuario, el frontend tendría que hacer una petición adicional por cada usuario. Por lo tanto, si una empresa tiene 10 trabajadores, el frontend tendría que hacer 10 peticiones HTTP adicionales, además de la petición inicial para obtener la organización.
